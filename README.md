# temu-orders (Chrome Extension)

## Load unpacked

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `src/` folder

## Use

1. Open a Temu tab (temu.com) and make sure you're logged in
2. Click the extension icon
3. Click **Fetch & Show Overlay**

## Notes

- Some sessions may require a changing header (`x-phan-data`). If requests fail, copy it from DevTools → Network and paste it into the popup.
